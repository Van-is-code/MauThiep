# Thiệp Cưới - Wedding Invitation

## Cách chạy

### Phương pháp 1: Sử dụng Node.js (Khuyến nghị)

1. Mở Terminal/Command Prompt tại thư mục này
2. Chạy lệnh:
   ```bash
   npm start
   ```
   hoặc
   ```bash
   node server.js
   ```

3. Mở trình duyệt và truy cập: http://localhost:3000

### Phương pháp 2: Mở trực tiếp file HTML

Đơn giản nhấp đúp vào file: `jmiiwedding.com/dathaaaaa/index.html`

**Lưu ý:** Một số tính năng có thể không hoạt động đúng khi mở trực tiếp file HTML do hạn chế CORS của trình duyệt.

## Cấu trúc thư mục

```
├── jmiiwedding.com/
│   └── dathaaaaa/
│       └── index.html          # Trang thiệp cưới chính
├── w.ladicdn.com/              # Thư viện CSS/JS
├── server.js                   # HTTP server
└── package.json                # Cấu hình Node.js
```

## Dừng server

Nhấn `Ctrl+C` trong Terminal để dừng server.
